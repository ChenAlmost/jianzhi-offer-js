## 题目描述
求 `1+2+3+...+n`，要求不能使用乘除法、for、while、if、else、switch、case 等关键字及条件判断语句（A?B:C）。

## 答案

```javascript
function Sum_Solution(n) {
  // write code here
  return ((1 + n) * n) / 2;
}
```
